DiagramEditor (Default Config)

The DiagramEditor component wired with default config only — no custom toolbar, no manual event listeners, no hand-rolled controls. Everything (toolbar, tool palette, color selects, font/size selects, undo, prompts) is handled by the component.

Navigate Fast

Hold Space + pointer drag to pan. Use mouse wheel or trackpad pinch to zoom, then click Fit all to quickly reframe your diagram.

Create Shapes

Pick a shape tool, drag to place, then resize with corner/edge handles. Use the rotate handle on the right side and hold Alt for angle snapping.

Select Multiple

Use Shift+drag for marquee selection. Use Cmd/Ctrl+click to toggle nodes in/out of selection, then align/distribute from the top toolbar.

Keyboard Workflow

Press Enter to edit text on selected nodes. Use Cmd/Ctrl+C, X, V, Z, Shift+Z/Y, and Delete/Backspace. Press Esc to finish in-progress drawing.
