This page is for end users of applications built with the diagrams editor.

## Pointer interactions

- Click on empty canvas: clear selection.
- Click a shape: select it.
- Shift + click shape: add or remove from selection.
- Drag selected shape(s): move.
- Drag corner/edge handle: resize.
- Drag rotation handle: rotate.
- Alt during rotation: snap rotation to fixed angle increments.
- Drag on empty canvas: rectangle selection.
- Shift during rectangle selection: keep current selection and add intersecting nodes.

### Connection editing

- Select a connection and drag control points to adjust path.
- Alt + click a segment point area: insert a connection point.
- Alt + click an existing internal point: remove that point.
- Drag connection ends/anchors to reconnect.

## Keyboard shortcuts

- Delete or Backspace: delete selected nodes.
- Ctrl/Cmd + C: copy.
- Ctrl/Cmd + X: cut.
- Ctrl/Cmd + V: paste.
- Ctrl/Cmd + Z: undo.
- Ctrl/Cmd + Shift + Z or Ctrl/Cmd + Y: redo.
- Ctrl/Cmd + A: select all.
- Escape: clear selection.
- Enter: edit selected node text.
- Ctrl/Cmd + Enter: apply text in text editor mode.

## Tips

- Use layer visibility controls to focus on parts of complex diagrams.
- Use fit actions after major edits to reframe the viewport.
- Keep related nodes aligned before adding connections for cleaner routing.
