# @samatawy/diagrams

Browser-first TypeScript primitives for diagram data, graph editing helpers, and viewport math.

This package intentionally starts small. It gives browser applications a typed model layer they can use under SVG, Canvas, WebGL, or framework-specific renderers without taking a dependency on any one rendering stack.
